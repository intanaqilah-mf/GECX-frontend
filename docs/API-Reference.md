# API Reference

The backend is built with FastAPI.

## Base URL
Default development URL: `http://localhost:8000`

## Core Modules
- **Home**: General dashboard and account overview.
- **Cards**: Management of physical and virtual cards (Activation, Details).
- **Applications**: New product applications.
- **Notifications**: System and transactional alerts.
- **Payments**: Money transfers and bill payments.
- **Devices**: Device management and security.
- **Push**: Push notification service.

## Endpoints
- `GET /`: Health check and service status.
- Refer to `/docs` on the running server for the full Swagger UI documentation.
